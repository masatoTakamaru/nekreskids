配列はすべて連想配列とする

特別な事情がない限り、配列は何らかのキーを持つことで統一する。キーに特別な意味を持たせない場合は、番号を振っておく。

          

//NGパターン
$arrSome = ['value1', 'value2', 'value3'];
//OKパターン
$arrSome = [
    'key1' => 'value1',
    'key2' => 'value2',
    'key3' => 'value3',
];
$arrSome = [
    1 => 'value1',
    2 => 'value2',
    3 => 'value3',
];

単一のフィールドに配列を保存する場合はjson文字列を用いる

データが単純な場合はスラッシュ区切りでも良いけど、json文字列に統一しておいた方が連想配列をそのまま扱えるし、複雑なデータ形式でも対応できる。

          

//保存するとき
$fieldData = json_encode($arrSome);
//抽出するとき
$arrSome = json_decode($fieldData, true);
